<h1 align=center>Vuetify migration</h1>
<p align=center>Testing Vuetify components for Vue 3.</p>

## 🚀 Installation

Make sure to install the dependencies

```bash
yarn install
```

## 👩‍💻 Development

Start the development server on http://localhost:3088

```bash
yarn dev
```

## 🏭 Production

Build the application for production:

```bash
yarn build
```
